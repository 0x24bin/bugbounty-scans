GSA BOARD OF CONTRACT APPEALS

Archive   Page

Contents

The Archive Page was created to preserve all of the appeal and bid protest decisions issued between October 1, 1992, when the Board started publishing its decisions on a Bulletin Board System (BBS), and September 30, 1996, when the GSBCA Home Page was initiated.  The Archive Page does not contain any travel, relocation, or transportation rate cases.  These decisions are only available on the GSBCA Home Page (www.gsbca.gsa.gov). 

Documents on the following pages are in two formats: HTML (Text) and WordPerfect 5.1 (in a self-extracting zip file).

Text files may be searched, displayed, printed, and downloaded easily.  When viewing cases in the text format, please use the "back" key on your browser to return to the index page.

WordPerfect 5.1 files are compressed into self-extracting zip files with a .exe extension. To extract the WordPerfect 5.1 file, itself, save the zipped file locally on your PC.  Double-click on the file in order to perform the self-extraction.  The extracted file will have the same filename with a numeric extension instead of .exe.  The numeric extension may be changed to .wpd or .doc in order for it to be opened in WordPerfect or Microsoft Word, respectively, by double-clicking on the file.  After the WordPerfect 5.1 file has been extracted, the original file with the .exe extension may be deleted.
