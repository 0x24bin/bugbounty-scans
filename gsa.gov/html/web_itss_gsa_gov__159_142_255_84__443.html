FY17 PI-4 Is Planned to be Released on Oct. 13

System Announcement PI-4 is tentatively planned to be released on Oct. 13, This release contains updates to Post Awards Collaboration (PAC) and Program Incremental Improvements (ITSS Funding Order Validation and Chronology). For more details on What's Changing this PI's Release access the following link: https://web.itss.gsa.gov/itss/v41_helpdocs.nsf/LUindex/What'sChangingInPI-4Summary

FY17 PI-3 Release

System Announcement On July 14th, 2017, the FY17 PI-3 Release was deployed. This release contains updates to Post Awards Collaboration, OMIS Funds Search, TOS ECF, ITSS Funding, PIID and FPDS. For more information on the details of the changes, please access the help document : https://web.itss.gsa.gov/itss/v41_helpdocs.nsf/LUindex/What'sChangingInPI3Summary
