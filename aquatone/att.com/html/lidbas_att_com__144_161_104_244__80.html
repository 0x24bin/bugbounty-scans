Launch Icon

Use the icon on the right to launch (run) the AT&T LIDB/AS TnMgr (TN Manager) Web application. See further discussion on all topics, including those below, on the FAQ (Frequently Asked Questions) page.

Java

Before running the LIDB/AS application the first time, Java should be installed on your desktop. Java JRE version 6.0 or later is required to run the LIDB/AS application. To install Java, see this item in the FAQ, and the downloads page.

Desktop Shortcuts/Icons

The first time you run TnMgr, it will create a shortcut icon on your desktop and a Start menu entry (under LIDB-AS). You may run TnMgr subsequently without returning to this page. Running from any method or location (desktop icon, Start menu, or website) is identical.

Automatic New Releases

Whenever a new version of TnMgr is released, it will automatically be installed the next time you run the application. Your system administrator may configure the system to prompt you first.

Online Help and Support

Online help is available here, and from the Help menu in any application. For a PDF copy of application help, see the downloads page. The telephone number and e-mail address for support appear at the bottom of pages on this website.
