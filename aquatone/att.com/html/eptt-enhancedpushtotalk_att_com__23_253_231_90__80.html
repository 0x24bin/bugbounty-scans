Corporate Administration Tool (CAT)

Corporate Administration Tool (CAT)

The CAT Training Session has been designed for the Administrator to get the most out of Enhanced-PTT and showcase its powerful user intuitive Web-based Interface.

This 90 minute instructor-led and live demo training session will teach participants how to fully leverage this tool.

The session will cover key features such as:

  • Features & Benefits of CAT
  • How to access CAT via Premiere Portal
  • Live CAT simulation/navigation demo
  • Pushing/sync & editing subscriber names
  • Assigning supervisor override demo
  • Benefits of using sub-lists
  • Add and delete external contacts
  • Support for advanced CAT features
  • Resources to FAQs

Integrated Dispatch Tool

Integrated Dispatch Tool

The Integrated Dispatch offering of Enhanced-PTT is a PC user intuitive application that enables company dispatchers/telecomm mangers to easily view, track and directly communicate with Subscribers.

This 90 minute instructor-led and live demo training session will teach participants how to fully leverage this tool.

The session will cover key features such as:

  • Record conversations
  • Send instant personal alerts
  • Dispatch call to 1-1, groups or ad-hoc groups
  • Friendly UI and PC application
  • Talk Group Scanning
  • User Enhanced-PTT Presence Status
  • Map location
