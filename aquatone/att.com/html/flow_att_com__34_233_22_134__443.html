  • Prototype

    Add Devices, collect sensor data via Streams, test Triggers, and iterate before distributing.

  • Organize

    Create and access groups of devices with Collections and categorize types of devices with Tags.

  • Go to Market

    Scale from a single device to millions with the click of a button using the Distribution tools.

  • Device Data Streams

    Collect and visualize time-series sensor or application data with Streams.

  • API Key Management

    Manage access to device stream data using API Keys, which provide POST, GET, PUT, DELETE level controls.

  • M2X Context

    Utilize sophisticated permissions based data management and routing tools to keep your data secure.

  • Event Triggers

    Set device Triggers to send an alert when a specific event occurs (e.g. a change in temperature or a drop in volume).

  • Trigger log

    Monitor device stream Trigger activity and on/off states.

  • Dashboards

    Visualize device Stream, Trigger and Location data with Dashboards.

  • Widgets

    Customize and add new Widgets to Dashboards and embed them in third party applications using simple auto-generated code.

  • Example queries

    Start pushing data to your device from day one. The Cheatsheet provides inline cURL commands that are specific to your device.

  • Great Documentation

    Take your development a step further with powerful tools outlined in our simple API Documentation.

  • Public Devices

    Make your device data available via public API endpoints for others to consume, mashup, or build on top of.
