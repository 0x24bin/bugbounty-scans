Landfill: The Bugzilla Test Server

About Landfill

Landfill is a home for test installations of the Bugzilla bug-tracking system. If you are evaluating Bugzilla, you can use them to try it out. They're also useful if you are a developer and you want to try to reproduce a bug that somebody has reported.

Test Installations

Note: This is a testing and demonstration site, so do not enter any personal information or anything that is considered private or security sensitive into any of these Bugzillas. Other than that, feel free to do anything to these installations! They are yours to play with.

Unfortunately it's currently not possible to experience what it's like to administer and customise Bugzilla without installing your own test system. See the instructions here.

Bugzilla Trunk

A demo installation running the "trunk" (meaning, very latest and unstable) Bugzilla code. This installation runs on MySQL. Install offline.
